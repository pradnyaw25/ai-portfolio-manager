# Brand assets

Source assets for off-site branding (not served by the site).

| File | Use | Size |
|------|-----|------|
| `x-avatar-light.png` / `x-avatar-dark.png` | X / social profile picture | 400×400 |
| `x-banner-light.png` / `x-banner-dark.png` | X profile header | 1500×500 |

Pair light-with-light or dark-with-dark. Site logo/favicon live in `public/`
(`logo.svg`, `favicon.svg`); the social preview card is `public/og-cover.png`.
Regenerate with the Pillow scripts used in the branding PR.
